# Azure VM Automation Dashboard (Frontend)

## Run Locally

1. Install Node.js (includes npm).
2. In the project folder:

```bash
npm install
npm run dev
```

## Notes

- Mock data only.
- Authentication is mocked via React context.
- VM actions call placeholder service functions in `src/services/vmService.ts`.
- Replace TODOs with real backend integrations when APIs are ready.
